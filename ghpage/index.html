Try PikPix

PikPix is a powerful command-line tool to convert and compress images with ease. Explore our features and learn how to get started.

Features

  • Convert images to various formats including JPEG, PNG, WebP, and TIFF.
  • Easily convert multiple images in a directory with a single command.
  • Reduce file sizes with adjustable compression levels.
  • Metadata is stripped by default to lower file size.
  • Ensures unique file names to prevent conflicts.

Get PikPix

Install PikPix globally to use it from anywhere on your system:

npm install -g pikpix

Or install it locally in your project directory:

npm install pikpix

To use PikPix, you need to have Node.js installed on your system.

Usage

To convert a single image file:

pikpix --input <input_file> --output <output_file> --format <format> [--compression <level>]

To convert all images in a directory: 

pikpix --input <input_directory> --output <output_directory> --format <format> [--compression <level>]

Examples

Convert a single image from PNG to JPEG with compression level 80: 

pikpix --input image.png --output image.jpg --format jpg --compression 80

Convert all WebP images in the `input_directory` to JPEG format with compression level 80: 

pikpix --input input_directory --output output_directory --format jpg --compression 80
